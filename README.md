### Awesome CV

Forked from [**Awesome CV**](https://github.com/posquit0/Awesome-CV). Awesome CV is LaTeX template for a **CV(Curriculum Vitae)** or **resume** or **cover letter** inspired by [Fancy CV](https://www.sharelatex.com/templates/cv-or-resume/fancy-cv). It is easy to customize your own template, especially since it is really written by a clean, semantic markup.
